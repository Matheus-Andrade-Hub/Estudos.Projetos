>VIEW MONITOR

Para ligar e desligar a camera do monitor da nave.

>SWITCH [Player name]

Para trocar a visao para um jogador no monitor principal (da para fazer isso no painel da nave, o que e bem mais pratico.)

>PING [Nome do radar]

Para fazer um Radar Booster emitir som, muito útil para distrair inimigos que estão em volta do Radar Booster.

>TRANSIMIT [mensagem]

Para transmitir uma mensagem a distancia para outros jogadores usando o Translator Signal.

>SCAN

Com este comando voce consegue ver quanto de créditos totais tem na lua em que voce está.
